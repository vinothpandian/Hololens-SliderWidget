<h2><a id="user-content-unity-asset-package" class="anchor" href="#unity-asset-package" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Unity asset package</h2>

<h2><a id="user-content-getting-started" class="anchor" href="#getting-started" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Getting Started</h2>
<h3><a id="user-content-importing-assets" class="anchor" href="#importing-assets" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Importing Assets</h3>
<ul>
<li>Download the latest <a href="https://github.com/Microsoft/HoloToolkit-Unity/tree/master/External/Unitypackages">Holotoolkit </a> and <a href="https://github.com/vinothpandian/Hololens-SliderWidget/tree/master/Release">Hololens-Slider</a> Unity asset packages.</li>
<li>Right click the <strong>Assets</strong> folder in the <strong>Project</strong> panel.</li>
<li>Click on <strong>Import Package &gt; Custom Package</strong>.</li>
<li>Navigate to the project files you downloaded and click on <strong>Holotoolkit-Unity-vX.unitypackage</strong>.</li>
<li>Do the same as above and now select <strong>Hololens-Slider.unitypackage</strong></li>
<li>Now you have imported all the necessary assets to use the Hololens slider widget</li>
</ul>
<h3><a id="user-content-setting-up-the-scene-for-hololens-development" class="anchor" href="#setting-up-the-scene-for-hololens-development" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Setting up the scene for Hololens development</h3>
<ul>
<li>Select <strong>File &gt; Save Scenes</strong> and save your current scene</li>
<li>In menu bar select <strong>HoloToolkit &gt; Configure &gt; Apply HoloLens Project Settings</strong> and choose <strong>Apply</strong></li>
<li>Unity will prompt for a  <strong>Project reload</strong>. Choose <strong>Yes</strong></li>
<li>Wait till unity reloads the project</li>
<li>In menu bar select <strong>HoloToolkit &gt; Configure &gt; Apply HoloLens Scene Settings</strong> and choose <strong>Apply</strong></li>
<li>Now you have successfully setup the Unity project for HoloLens development</li>
</ul>
<h3><a id="user-content-using-the-holotoolkit-input-and-cursor-manager" class="anchor" href="#using-the-holotoolkit-input-and-cursor-manager" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Using the Holotoolkit Input and Cursor Manager</h3>
<ul>
<li>In Project panel,</li>
<li>Type "inputmanager". Select the <strong>prefab "InputManager"</strong> and drag-drop it to the Scene.</li>
<li>Type "cursorwith". Select the <strong>prefab "CursorWithFeedback"</strong> and drag-drop it to the Scene.</li>
<li>Now you have successfully setup the Input and Cursor Manager of Holotoolkit</li>
</ul>
<h3><a id="user-content-using-the-hololens-slider-widget" class="anchor" href="#using-the-hololens-slider-widget" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Using the Hololens Slider widget</h3>
<ul>
<li>In Project panel,</li>
<li>Type "hololens-slider". Select the <strong>prefab "Hololens-Slider"</strong> and drag-drop it to the Scene.</li>
<li>Select the GameObject that should be connected with the Hololens Slider.</li>
<li>In the Inspector Panel, choose <strong>Add Component</strong></li>
<li>Type "sliderconnection" and click on the "<strong>Slider Connection" script</strong> to add it.</li>
<li>Drag the <strong>Hololens-Slider</strong> from the Scene and drop it on the Slider option in <strong>Slider Connection</strong> script</li>
<li>Now in the GameObject script you can use <strong>SliderConnection.Value</strong> to get the Slider widge current value</li>
</ul>
<h3><a id="user-content-building-and-deploying-the-hololens-application-in-hololens-emulator" class="anchor" href="#building-and-deploying-the-hololens-application-in-hololens-emulator" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Building and Deploying the HoloLens Application in Hololens Emulator</h3>
<ul>
<li>Select <strong>File &gt; Build Settings</strong> and select <strong>Windows Store</strong> option.</li>
<li>Choose <strong>Switch platform</strong> if not chosen already. Also select <strong>Unity C# Projects</strong> in Debugging.</li>
<li>Click <strong>Build</strong>.</li>
<li>Create a <strong>New Folder</strong> named "App".</li>
<li>Single click the <strong>App Folder</strong>.</li>
<li>Press <strong>Select Folder</strong>.</li>
<li>When Unity is done, a File Explorer window will appear.</li>
<li>Open the <strong>App</strong> folder in file explorer.</li>
<li>Open the generated <strong>Visual Studio solution</strong></li>
<li>Using the top toolbar in Visual Studio, change the target from Debug to <strong>Release</strong> and from ARM to <strong>X86</strong>.
<ul>
<li>Click on the arrow next to the Device button, and select <strong>HoloLens Emulator</strong>.</li>
<li>Click <strong>Debug -&gt; Start Without debugging</strong> or press <strong>Ctrl + F5</strong>.</li>
<li>After some time the emulator will start with your project.</li>
</ul>
</li>
</ul>
